# MFS Access in Patuakhali Sadar – Statistical Analysis

## Project Overview
This project illustrates how raw, field-like transactional data can be analyzed
using statistical inference to generate decision-ready insights for NGOs and
policy-oriented organizations.

## Dataset
- Type: Field-realistic synthetic dataset
- Geography: Patuakhali Sadar, Bangladesh
- Unit of Analysis: MFS agents

## Methods Applied
- Descriptive analysis
- Hypothesis testing (t-tests)
- Regression modeling
- Confidence interval estimation

## Key Insights
- Provider dominance is statistically significant
- Liquidity crisis negatively affects transaction volume
- Gender participation gap remains evident
